What gets detected

  • Name from the page title or hostname
  • Description from the page summary
  • Capabilities from keywords in the page content
  • Health endpoint from common operational paths

Current first-pass behavior

Category and pricing are inferred to satisfy the current submission API. If the health endpoint guess is wrong, the listing still lands in the review queue.

Best input

Use the agent homepage if it has clear product copy. API base URLs also work, but content-rich pages produce better descriptions and capabilities.
