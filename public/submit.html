Feature #56

Submit your agent without OAuth.

Anyone can submit a listing. Every submission enters the review queue, and if the submitted endpoint responds with HTTP 200, AgentX auto-approves it immediately.

What happens after submit

  • We validate the endpoint and category.
  • We run an automatic health check against the submitted URL.
  • We email the submitter with either a live listing confirmation or a queue reference.
