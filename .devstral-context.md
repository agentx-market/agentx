# AgentX Codebase Context (injected into every coding dispatch)

## Auth Pattern
- Session middleware sets `req.operatorId` (NOT req.session.operator)
- Check login: `if (!req.operatorId) return res.redirect('/auth/github');`
- OAuth routes: GET /auth/github, GET /auth/google
- API auth: `requireAuth` middleware checks session cookie OR X-AgentX-Key header
- db is already imported at top of server.js: `const db = require('./db')`

## Agents Table Schema
```
id INTEGER PRIMARY KEY, operator_id TEXT, name TEXT, description TEXT,
capabilities TEXT (JSON array), endpoint_url TEXT, pricing TEXT,
status TEXT DEFAULT 'pending', health_check_passed_at INTEGER (ms epoch),
health_endpoint_url TEXT, created_at INTEGER, updated_at INTEGER,
wallet_id TEXT, apiKeyHash TEXT
```
NO slug column (derive from name: `name.toLowerCase().replace(/ /g, '-')`)
NO category column on agents (categories use join tables: agent_categories + categories)
NO uptime_percent, rating, health_status, health_check_url columns

## HTML <head> Pattern (all static pages)
All public/*.html files already have OG meta tags and structured data in <head>.
BEFORE adding meta tags: grep for existing og:, twitter:, schema.org tags.
UPDATE existing tags — never add duplicates.

## Helper Scripts
- `node scripts/html-edit.js <file> <selector> <action> [value]` — DOM-based HTML editing (avoids exact-match failures)
  Actions: replace-inner, replace-outer, set-attr, append-child, prepend-child, remove, get-text, get-html
- `bash ~/web_search.sh "query"` — local SearXNG search
- `node --check server.js` — syntax check before restarting

## Common Pitfalls
- server.js is 1200+ lines — NEVER read it all. grep first, read 10-20 lines.
- Static HTML files have inline <style> blocks — don't confuse them with external CSS.
- The `db` module uses better-sqlite3 (synchronous). Queries return immediately, no await needed.
- EJS views use `<% include partials/navbar %>` NOT `<%- include("partials/navbar") %>` (old EJS syntax).
- Always restart after editing server.js: `pkill -f 'node.*server.js' ; sleep 1 ; cd ~/marco_web && nohup node server.js > /tmp/webserver.log 2>&1 &`
